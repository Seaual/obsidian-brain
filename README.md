# Obsidian Brain Map

> 自我认知复利 — 每次对话都在沉淀"我"是怎么运作的

## 是什么

一个关于"我"的认知 wiki —— 每次对话都在沉淀：大 w 是怎么思考的、什么是他的盲区、他的"道"是什么。

不是知识积累，是**自我认知复利**。Wiki 随对话越来越厚，对"我"的理解越来越深。

## 核心框架：用 / 体 / 道

| 层级 | 内容 | 例子 |
|---|---|---|
| **用** | 方法、技巧、工具 | MKG 的用法、Hermes Cron |
| **体** | 概念、模式、分类 | 第一性原理、自我认知复利 |
| **道** | 方向、假设、底层信念 | 系统简洁性 > 并行冗余 |

## 设计原则

- **不积累知识** — 积累的是"我"的运作模式
- **实时审计** — 每次对话后立即更新，不是沉淀后再看
- **零废话** — 只有引用和推演，没有解释和教学
- **Count 作为理解深度** — Count 越高，道层越清晰
