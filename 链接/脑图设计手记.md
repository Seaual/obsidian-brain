# 脑图设计手记

## 2026-04-18 重大发现：洗地律师

**场景**：与 AI 讨论 Agent 自我进化时，AI 提出"System 1 是主人，System 2 监控和控制"

**反应**：我的 System 1 在 0.1 秒内判断"不对"，然后 System 2 开始找理由

**发现**：System 2 不是主人。真正的决策在 System 1 里已经完成，System 2 只是在为已做的判断辩护——像个洗地律师。

**道的暴露**：我以为"我"在控制思考，实际上"我"是事后找理由的那个。

**这让我意识到**：道不是被想到的，是被看见的。需要停下来、回头看，才能发现。

## 三层结构（用/体/道）

每条认知同时存在于三层，count 代表被激活的总次数。

```
         ┌─────────────────────────────┐
         │         用（表层）            │  ← 此刻在说什么
         │   "我通过X，达到Y"            │
         └──────────────┬────────────────┘
                        │ 抽象化
         ┌──────────────▼────────────────┐
         │         体（中层）             │  ← 说出来背后的结构
         │   "我理解X的本质是Y"           │
         └──────────────┬────────────────┘
                        │ 提炼
         ┌──────────────▼────────────────┐
         │         道（深层）              │  ← 底层假设，自己不可见
         │   "我相信X是不言自明的真理"     │
         └─────────────────────────────┘
```

## 三层表达公式

| 层 | 公式 | 例子 |
|---|------|------|
| **用** | "我通过 [X]，[做/达到/实现] [Y]" | 我通过 MKG，做论文概念提取 |
| **体** | "我理解 [X] 的本质是 [Y]" | 我理解缠论的本质是走势终完美 |
| **道** | "我相信 [X] 是不言自明的" | 我相信优化一定有最优解 |

## count（神经元粗壮度）

独立于状态层，表示历史上被激活的总次数。

```
count = 0  → 从未被激活
count = 1-3 → 道表层，刚激活
count = 4-8 → 体层，反复沉淀
count ≥ 9  → 用层，高度熟练
```

count 不决定层，层由"此刻的认知形态"决定。

## 认识思维局限

| 层级 | 盲区类型 |
|------|---------|
| 用 | 带宽局限：同时处理不了 7±2 个信息块 |
| 体 | 结构局限：只能用已有框架理解新事物 |
| 道 | **范式局限：自己的底层假设不可见** |

道的盲区最隐蔽，也最致命。

## 认识自己

- **用** → 我现在在想什么，我的注意力消耗在哪
- **体** → 我用什么框架思考，我擅长什么模式
- **道** → 我相信什么是理所当然的，我的底层信念是什么

## 映射到东方哲学

- 用 = 我此刻说出来的话
- 体 = 我说出来的话背后的结构
- 道 = 我说不出的话，但它在操控我说什么
