# 脑图设计参考资料

## 1. Metacognition（元认知）

**来源**: [Wikipedia - Metacognition](https://en.wikipedia.org/wiki/Metacognition)

> "Metacognition is an awareness of one's thought processes and an understanding of the patterns behind them."

元认知两个核心组成：
- **认知 Conception** — 我用什么框架思考
- **认知 Regulation** — 我如何监控和调整自己的思考

**映射到我们的框架**：

```
用/体/道本身就是元认知的三个层级
- 用 = 认知 Regulation（此刻在监控）
- 体 = 认知 Conception（已形成的框架）
- 道 = 框架的底层假设（看不见，需要外部反馈）
```

## 2. Predictive Processing（预测加工）

**来源**: [arXiv/学术文献]

> 脑是大脑是预测机器，通过最小化预测误差来更新内部模型。

核心：
- 我们的"道" = 预测加工的 top-down prior（先验）
- "道不可见" 因为它是产生预测的源头，不是一个可观察的输出
- 你永远用自己的"道"来解释一切，所以你看不到它

**映射**：
```
道 = 你大脑中的 prior
你用 prior 产生预测（对世界的理解）
你只能通过预测误差（别人与你的冲突）来间接知道 prior 的存在
```

## 3. Consciousness Models（意识模型）

来源: [Wikipedia - Consciousness - Models section](https://en.wikipedia.org/wiki/Consciousness#Models)

| 模型 | 核心观点 | 与用/体/道的关联 |
|------|---------|----------------|
| **Global Workspace Theory** | 信息广播到全脑意识 | 用 = 信息的全局广播 |
| **Integrated Information Theory (IIT)** | 意识 = 整合信息 Φ | 体 = 整合后的结构 |
| **Attention Schema Theory** | 脑对注意力建立内部模型 | 元认知的核心机制 |
| **Entropic Brain Theory** | 意识程度与大脑信息熵相关 | 道 = 低熵的核心规律 |

## 4. Cognitive Bias（认知偏差）

**来源**: "Addressing Cognitive Bias in Systems Engineering Teams" (INCOSE 2020)
DOI: 10.1002/j.2334-5837.2020.00721.x

> "Cognitive bias is a general issue in human decision making. It is easy to assume there is less bias in engineering... however, human decision making is almost always affected by bias."

**关键洞察**：
- 偏差存在于"道"的层面——不是有意识的推理过程
- 系统工程中，识别认知偏差是突破思维局限的核心
- 偏差种类：确认偏误、可得性启发、锚定效应……

**映射到我们的框架**：
```
道的盲区 = 认知偏差的集合
"我我相信 X 是不言自明的" = 最纯粹的认知偏差入口
```

## 5. Neurocognitive Model of Ideological Thinking

**来源**: "A neurocognitive model of ideological thinking" (Zmigrod, 2021)
DOI: 10.1017/pls.2021.10

> "There are neurocognitive antecedents to ideological thinking: the brain's low-level neurocognitive dispositions influence its receptivity to ideological doctrines."

核心观点：
- 意识形态（道的一部分）有神经认知前因
- 低层的神经认知倾向影响对意识形态的接受度
- 暴露在某种意识形态下会反过来塑造认知系统

## 6. Self-Awareness (自我意识)

**来源**: [Wikipedia - Self-awareness]

> "Self-awareness is the capacity for introspection and the ability to recognize oneself as a separate individual."

关键区分：
- **自我意识** = 知道自己是谁
- **元认知** = 知道自己在怎么思考

我们的框架覆盖两者：
```
用 = 此刻的自我意识
体 = 我对自己的思维模式的理解
道 = 我对自己底层信念的无意识（看不到）
```

## 整合：道的可见化方法

根据以上研究，"道"能被看见的几种路径：

1. **预测误差** — 当你的预测失败，道才显现（别人与你的理解冲突）
2. **跨范式对照** — 用完全不同的框架来看同一问题（PhD 的跨学科视角）
3. **外部反馈** — 别人直接指出你的底层假设（辩论、导师、合作伙伴）
4. **极限情境** — 当你被逼到角落，旧的道崩溃，新的显现（创伤/重大失败）
5. **刻意反思** — 元认知训练可以部分看到体层，但道的最深处仍不可直接触及

## 7. First Principles Thinking（第一性原理思维）

**来源**: [Wikipedia - Abstraction](https://en.wikipedia.org/wiki/Abstraction)

> "Abstraction is the process of generalizing rules and concepts from specific examples... The result of the process, an abstraction, is a concept that acts as a common noun for all subordinate concepts."

核心：
- First Principles = 从具体事例中**提取**最底层的公理
- Abstraction = 从具体中**泛化**规则
- 这就是从"用"到"体"到"道"的提炼过程

**映射到我们的框架**：

```
具体事例 ──abstraction──→ 体（规则/概念）
具体事例 ──first principles──→ 道（公理/底层假设）

"用"是具体操作
"体"是抽象规则
"道"是最底层的公理
```

First Principles Thinking 就是不断追问"为什么"，直到无法再问"为什么"为止。那个无法再问的点 = 道。

## 8. Dual Process Theory（双过程理论）

**来源**: [Wikipedia - Dual process theory](https://en.wikipedia.org/wiki/Dual_process_theory)

> "A dual process theory provides an account of how thought can arise in two different ways: an implicit (automatic), unconscious process and an explicit (controlled), conscious process."

关键人物与历史：
- ** Spinoza** (1632-1677) — 区分 passion 和 reason
- **William James** (1842-1910) — 区分 associative thinking 和 true reasoning

James 的理论：
- **Associative knowledge** = 只有过去经验，"only reproductive"
- **True reasoning** = 能克服"未曾经历过的情境"，像地图导航

**映射到我们的框架**：

| 双过程理论 | 威廉·詹姆斯的描述 | 映射到我们 |
|-----------|-----------------|-----------|
| System 1 (快/自动/无意识) | Associative thinking | 道（底层自动运行）|
| System 2 (慢/控制/有意识) | True reasoning | 用（在意识中控制运行）|

但更有趣的是：

```
道 → 产生 System 1 的自动反应
用 → 是 System 2 在监控和干预 System 1
体 → 是从反复的"用"中沉淀出的结构
```

**关键洞察**：你的"道"在操控 System 1，而你以为自己在用 System 2 做理性决策。

## 9. Cognitive Bias（认知偏差）

**来源**: [Wikipedia - Cognitive Bias](https://en.wikipedia.org/wiki/Cognitive_bias)

> "A cognitive bias is a systematic pattern of deviation from norm or rationality in judgment."

核心：
- 认知偏差是**系统性的**偏离理性的模式
- 偏差让你构建自己的"主观现实"，而不是客观输入
- 认知偏差有时候是**适应性的**（有用的）

**与道的关联**：

道=底层假设，认知偏差=底层假设的运作方式

你看不到自己的道，因为道通过认知偏差来运作——偏差是"道"的表现形式，是"道"产生判断结果的过程。

主要偏差类别（可能对应不同类型的道）：

| 偏差类型 | 例子 | 可能的道的来源 |
|---------|------|--------------|
| 确认偏误 | 只看支持已有信念的信息 | 我的道：我相信X |
| 后见之明 | 事后认为"早就知道" | 我的道：我能预测 |
| 锚定效应 | 被第一个信息过度影响 | 我的道：最初的认知 |
| 损失厌恶 | 对损失的敏感度高于收益 | 我的道：稳定比机会重要 |
| 可得性启发 | 用最容易想到的例子判断 | 我的道：熟悉=真实 |

## 整合：道的可见化方法（更新版）

基于以上研究，"道"能被看见的路径：

1. **预测误差**（Predictive Processing）— 当预测失败，道才显现
2. **跨范式对照**（First Principles）— 用完全不同的框架来看同一问题
3. **外部反馈**（Cognitive Bias 识别）— 别人直接指出你的偏差
4. **极限情境** — 旧的道崩溃，新的显现
5. **刻意练习 System 2**（Dual Process Theory）— 用"用"来监控和干预 System 1
6. **抽象化训练**（Abstraction）— 从具体中提炼公理，追问"为什么"直到无法再问
